# VRCMods
 Mods for VRChat

## Dependencies
 These mods use [ReMod.Core](https://github.com/RequiDev/ReMod.Core)
 - URLTools
 - ProgramLauncher
 - MLConsoleViewer
 - ShortCuts
 - QMPet
 - OldMate

## Icons
 These mods use Icons from [Noun Project](https://thenounproject.com/)
 - URLTools
 - ProgramLauncher
 - MLConsoleViewer
 - ShortCuts
 - OldMate
